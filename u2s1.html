    2. Complements Rule

    What do you notice about P(\(B\)) and P(\(\overline{B}\))? \(P(B)+P(\overline{B})=\frac{4}{9}+\frac{5}{9}=1\)

    What do you notice about P(\(R\)) and P(\(\overline{R}\))? \(P(R)+P(\overline{R})=\frac{3}{9}+\frac{6}{9}=1\)



    Probability Rules for Complements

    1. \(P(A)+P(\overline{A})=1\)
    2. \(P(A)=1-P(\overline{A})\)
    3. \(P(\overline{A})=1-P(A)\)

    Some Special Probabilities

    Impossible Event: probability of 0

    Certain Event: probability of 1

    Unusual Event: probability is very low

  2. You roll a fair 6-sided die

    1. List the Sample Space: \((S)=\{1,2,3,4,5,6\}\)     \(n(S)=6\)
    2. What is the probability of:
      1. rolling an odd number greater than 1?

        \(P(E)=\frac{n(E)}{n(S)}\)

        \(E=\{3,5\}\)

        \(n(E)=2\)

        \(P(E)=\frac26=\frac13\)

      2. not rolling an odd number greater than one?

        This is a good example when to use the Rule of Complements. The problem above is the complement of this problem.

        P(not rolling odd greater than 1) = 1 - P(rolling odd greater than 1) \(=1-\frac{2}{6}=\frac{4}{6}\)

        \(=\{1,2,4,6\} ; n(E)=4\)

      3. rolling a value greater than 6? \(\frac{0}{6} ; E=\{ \}\) impossible
      4. rolling a value between 1 and 6, inclusive? \(\frac{6}{6} ; E\{1,2,3,4,5,6\} \) certain

    4. Fundamental Counting Principle:

    For a sequence of two events in which the first event can occur in m ways and the second event can occur in n ways, the events together can occur in m*n ways. This rule can be extended to any number of events occurring in sequence.
